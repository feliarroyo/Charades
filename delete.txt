**Selector:**
[X] -> Carga los categorybuttons correspondientes
[X] -> Al clickear, mandan la categoría a CustomEditor, y se carga dicha escena
[X] -> Hay un botón volver para regresar.

**Editar: **
[ ] -> archivo ya fue creado, hay que guardar el dato del nombre original
[ ] -> cargar ícono utilizado actualmente

Terminar:
[X] -> pongo nuevos datos en archivo (nombre original)
[X] -> actualizo el nombre (si es necesario y habiendo chequeado si es posible)
[X] -> salgo de la escena

Borrar categoría:
[ ] -> borro el archivo

Borrar enunciado: 
[ ] -> Por cada renglón hay un botón cruz
[ ] -> Apretarlo busca el enunciado correspondiente de la categoría y lo borra
[ ] -> Actualizar lista de enunciados
